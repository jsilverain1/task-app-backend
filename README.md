# Task App — Spring Boot Backend

A RESTful API backend for a full stack task management application built 
with Spring Boot, Spring Security, and PostgreSQL. Features JWT authentication, 
BCrypt password hashing, and user-owned task management.

## Features
- User registration and login with JWT authentication
- BCrypt password hashing — passwords are never stored in plain text
- Protected API endpoints — users can only access their own data
- Full task CRUD operations
- Automatic database schema generation via Hibernate
- Deployed and live on Railway

## Tech Stack
- Java 21
- Spring Boot 3.5
- Spring Security
- Spring Data JPA / Hibernate
- PostgreSQL
- JWT (jjwt 0.12.3)
- Maven
